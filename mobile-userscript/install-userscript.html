安装瀑光 FlowLens

当前统一版本：v1.4.6。电脑端与手机端入口分离，核心功能保持同步。推荐电脑端安装油猴版，手机端安装手机整合版；需要更强下载和跨源能力时再使用扩展版。

💻 电脑端：Tampermonkey 油猴版（推荐）

一键安装/更新电脑油猴版 v1.4.6

适合桌面 Edge、Chrome、Firefox 安装 Tampermonkey 后使用。电脑端不要安装手机整合版脚本。

📱 手机端：Tampermonkey 油猴版

一键安装/更新手机整合版 v1.4.6

点完后，Tampermonkey 正常会弹出安装页，点“安装”或“更新”即可。若仍显示旧版本，请先删除旧脚本，再用下方地址重新安装。

🧩 电脑端：Edge / Chrome 扩展版（备用）

扩展版最新版本：v1.4.6。如果需要使用扩展版，请在 Edge / Chrome 的扩展管理页开启“开发者模式”，选择仓库里的 flowlens-extension 目录并“加载解压缩的扩展”。更新后在扩展管理页点“重新加载”。

安装不弹出、仍显示旧版本或需要分开安装时点这里

电脑油猴版 URL：

https://fallen0909.github.io/flowlens/flowlens-desktop.user.js?v=1.4.6

手机整合版 URL：

https://fallen0909.github.io/flowlens/flowlens-mobile-all.user.js?v=1.4.6

也可以在 Tampermonkey 的“工具/Utilities”里使用“从 URL 安装”。

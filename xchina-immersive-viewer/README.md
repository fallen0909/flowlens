# 电脑端：瀑光 FlowLens 扩展

![瀑光 FlowLens 项目封面](../docs/assets/flowlens-hero.png)

这是瀑光 FlowLens 的电脑端 Edge / Chrome 未打包扩展版本。它会把网页里零散、分页、尺寸不一的图片收拢起来，变成一个全屏、顺滑、适合长时间浏览的沉浸式图片流。

很多套图网页的痛点都很具体：页面广告多，图片间距乱，下一页要反复点，缩略图太小，点开大图又会丢失浏览位置；遇到视频或类 GIF，还要在不同播放器和页面之间来回切。瀑光把这些碎片整理成一个干净的瀑布流视图，让你像刷相册一样看网页图片。

## 为什么有意思

![瀑光 FlowLens 图标](../docs/assets/flowlens-icon-source.png)

- 一键进入全屏图片流：不用在原网页的布局、广告、分页按钮里找图。
- 智能收集大图和媒体：支持图片，也支持 `mp4`、`webm`、`mov`、`m4v` 这类视频卡片。
- 瀑布流密集浏览：一次看到更多内容，适合快速筛选、回看和沉浸式阅读。
- 大图模式更像本地相册：方向键、滚轮、侧边箭头切换，`Esc` 关闭。
- 原尺寸查看和拖拽：点击图片/视频可在适应屏幕和 1:1 原尺寸之间切换，原尺寸下可拖动查看细节。
- 下载更省事：收集到的内容可以统一打包下载。
- 本地运行更安心：扩展只在你的浏览器里读取当前网页和明确适配站点的同套图页面，不上传图片。

## 支持页面

通用网页会收集当前页面已经存在、以及后续动态新增的有效大图和媒体。只有明确适配过的套图详情页才会触发多页抓取，避免对任意网站盲目扫分页导致网页卡死。

当前重点适配：

- `xchina.co/photo/*`
- `x.810114.xyz/photo/*`

## 安装

### 桌面 Edge / Chrome 扩展

1. 打开浏览器扩展页面：
   - Edge: `edge://extensions`
   - Chrome: `chrome://extensions`
2. 开启开发者模式。
3. 点击“加载解压缩的扩展”或“Load unpacked”。
4. 选择 `xchina-immersive-viewer` 文件夹。

## 使用

1. 打开支持的套图页面，例如：
   `https://x.810114.xyz/photo/id-.../1.html`
2. 点击右下角“瀑光”悬浮入口，或点击浏览器扩展图标。
3. 页面会打开全屏瀑布流，图片和视频卡片会按可读性自动排布。

常用操作：

- `G`: 打开或关闭瀑光
- `+`: 增加列数
- `-`: 减少列数
- `D`: 打包下载已收集内容
- `Home`: 回到顶部
- `End`: 跳到底部
- 点击图片/视频：进入大图预览
- `Esc`: 关闭大图，再次关闭瀑光
- `ArrowLeft` / `ArrowRight`: 切换上一张/下一张
- 鼠标滚轮：在大图模式切换上一张/下一张

瀑光不是爬虫工具，而是一个把“网页看图”变舒服的小工具。它关注的是当前浏览体验：少点跳转，少点干扰，多一点连续、清爽、能看细节的图片流。

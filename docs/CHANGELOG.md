# 变更日志

## 1.7.19 - 2026-06-26

- 新增大图 iOS 风格顺滑补丁，接管 lightbox 图片 `src` 替换，先后台预解码高清图，再平滑替换，减少缩略图切高清图时的闪烁和跳动。
- 强化大图媒体层过渡效果，覆盖稳定补丁中过度禁用动画的问题，恢复淡入、轻微缩放和 GPU 合成优化。
- 点击瀑布流卡片前预热当前图及相邻图片，提升首次打开和连续切换时的响应速度。
- 同步更新电脑端、手机端入口脚本、版本中心和 `version.json`。

## 1.7.3 - 2026-06-24

- 新增 Meitulu 多页套图补齐采集，支持 `item/编号.html`、`item/编号_2.html` 这类分页自动识别、预取和注入采集。
- 入口脚本改为统一加载共享源码，避免电脑端、手机端入口与 `src/` 源码不同步导致修复不生效。
- 同步更新安装页、版本中心和 `version.json`。

## 1.7.2 - 2026-06-22

- 修复紧凑工具栏 CSS 仍会隐藏全屏按钮的问题。
- 构建时校验全屏按钮存在且未被紧凑工具栏隐藏，避免后续更新再次遗漏。

## 1.7.1 - 2026-06-22

- 修复紧凑工具栏错误隐藏右上角全屏按钮的问题。
- 图片流优化：提升首屏优先级，并提前加载接近视口的图片。

## 1.2.1 - 2026-06-15

- 入口位置改用扩展级 `chrome.storage.local` 保存，拖动后的坐标会在所有网页共用。
- 加强右下角入口拖动逻辑，拖动时阻止默认按钮行为，鼠标移出按钮后仍能继续拖动并保存位置。
- 同步更新电脑端扩展、移动端油猴脚本和本机 Edge 加载目录。

## 1.2.0 - 2026-06-15

- 右下角入口支持拖动位置。
- 设置面板新增“圆形纯图标”入口模式，可隐藏“瀑光”文字。
- 新增设置持久化：列数、主题、自动滚动速度、自动全屏、视频预览开关。
- 新增诊断面板，显示当前页面模式、媒体数量、分页状态、失败页数、过滤/去重数量和关键设置。
- 新增图片/视频筛选、收藏 ZIP 下载和链接导出。
- 移动端 `flowlens.user.js` 同步重建。

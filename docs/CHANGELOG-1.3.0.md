# FlowLens 1.3.0 产品化升级

## 新增功能

- 新增扩展弹窗：点击浏览器工具栏图标可打开/关闭图片流、进入设置页、查看快捷键。
- 新增独立设置页：默认列数、主题、自动全屏、视频预览、隐藏入口、自动滚动速度、下载文件夹名。
- 新增快捷键说明面板：图片流内按 `?` 打开。
- 新增筛选快捷键：`1` 全部、`2` 图片、`3` 视频、`V` 循环切换。
- 新增选择模式：`S` 开关选择模式，点击图片/视频加入选择，`X` 清空选择。
- 新增选择下载：`Shift+D` 下载已选内容；没有选择时下载当前筛选可见内容。
- 新增链接复制：`E` 复制已选或当前可见链接。
- 新增浏览历史记录基础数据：记录当前页面标题、URL、数量、筛选和滚动位置，供后续继续浏览面板使用。

## 性能优化

- 对瀑布流卡片增加 `content-visibility`、`contain` 等低风险软虚拟化优化。
- 对离开视口较远的视频自动暂停，减少后台播放和解码压力。
- 大图模式切换时预加载临近图片，提升连续浏览流畅度。
- 保留原有瀑布流主逻辑，避免大拆导致原功能不稳定。

## 稳定性优化

- 继续加强知乎图片去重，按 `v2-xxx` 图片 ID 归并不同尺寸、不同参数的同图资源。
- 修复退出图片流后页面可能继续被锁定、无法上下滚动的问题。

## 工程化

- 新增 GitHub Actions 自动检查和打包 `flowlens-extension.zip`。
- 每次推送 master 后，会生成 Actions artifact，便于直接下载扩展包。
